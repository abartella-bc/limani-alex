# resizeButtonMenu

This is used to resize the button menu, in cases where the top nav have dynamic buttons which can be hidden.\
The component should always be wrapped inside parent element that has an `id=root`, all the pages in Gateway is wrapped with a top-level parent element that has an `id=root`.

## Parameters

<table>
    <thead>
        <tr>
            <th>Param</th>
            <th>Type</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>expanded</td>
            <td>`boolean`</td>
            <td>The expanded value from useMenuHandler.</td>
        </tr>
        <tr>
            <td>buttonId</td>
            <td>`string`</td>
            <td>The ID of the button to be clicked to show the menu.</td>
        </tr>
        <tr>
            <td>menuId</td>
            <td>`string`</td>
            <td>The ID of the Layer in which menu is displayed.</td>
        </tr>
    </tbody>
</table>

## Returns

`usePageMessages~Return` Resize the button menu, in cases where the top nav have dynamic buttons which can be hidden.

## Example

```js
import { useState } from 'react';
import { resizeButtonMenu } from '../utils/display';
import usePlatformData from '../hooks/usePlatformData';

const TestMenu = () => {
    const { data } = usePlatformData();
    const userData = {
        userName: data?.user.username,
    };
    const [expanded, setExpanded] = useState(false);
    resizeButtonMenu(expanded, 'testMenuBtn', 'testMenu');

    return (
        <div className={TestAccountMenu}>
            <button
                id='testMenuBtn'
                onClick={() => setExpanded(!expanded)}
                type='button'></button>
            {expanded && (
                <Layer id='testMenu' role='menu'>
                    <div id='user-profile'>
                        <span>{userData.userName}</span>
                    </div>
                </Layer>
            )}
        </div>
    );
};
```
